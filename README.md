Clone the repository:\n\n   ```markdown\n   git clone [https://github.com/NileshMete/attendance-calculator.git](https://github.com/NileshMete/attendance-calculator.git)\n   cd attendance-calculator\n   ```\n\n2. Install dependencies:\n\n   ```markdown\n   npm install\n   ```\n\n3. Run locally:\n\n   ```markdown\n   npm start\n   ```\n\n4.
